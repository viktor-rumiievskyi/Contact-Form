
# 📬 Contact Form

A simple and responsive **Contact Form** built with **HTML**, **CSS**. Ideal for collecting messages or inquiries on personal or business websites.

## ✨ Features

- 📥 Input fields for name, email, subject, and message
- ✅ Client-side validation
- 📱 Responsive layout for all devices

## 💡 Technologies Used

- HTML5
- CSS3

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/viktor-rumiievskyi/Contact-Form.git
cd Contact-Form
```


## 📁 Project Structure

```
Contact-Form/
│
├── index.html       # Contact form HTML
├── style.css        # Styling
└── README.md        # Documentation
```

## 🛠 Customization

- 🔧 Change form action URL in `index.html`
- 🎨 Modify styles in `style.css` to match your brand

## 🌟 Potential Improvements

- 🧪 Server-side validation
- 🧾 Save messages to a database
- 🛡️ Add reCAPTCHA or honeypot spam protection
- 📬 Email autoresponder functionality
